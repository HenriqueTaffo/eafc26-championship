# EAFC26 Championship

Projeto segmentado para GitHub Pages.

## Como subir no GitHub

1. Extraia este ZIP.
2. No repositório `eafc26-championship`, envie:
   - `index.html`
   - pasta `css`
   - pasta `js`
3. Faça commit.
4. Aguarde o GitHub Pages publicar.

## Estrutura

```text
index.html
css/styles.css
js/config.js
js/data.js
js/utils.js
js/api.js
js/standings.js
js/calendar.js
js/cups.js
js/transfers.js
js/events.js
js/players.js
js/forms.js
js/main.js
```
